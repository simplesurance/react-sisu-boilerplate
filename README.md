Quick start:

1. Install yarn [yarn](https://yarnpkg.com/en/docs/install) globally

2. Install npm packages in user and components folders
```
yarn
```

3. Run the webpack build:
```
yarn start
```

4. Open the url in your browser:
```
http://localhost:3000
```
